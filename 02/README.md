#Enable setting
###Enable vr mode 
Enter ```about:config``` at tab.
Set ```dom.vr.enabled;true```
###Enable fullscreen on Firefox mobile
Enter ```about:config``` at tab.
Set ```full-screen-api.allow-trusted-requests-only;false```
